# 项目简介
Linux 环境下基于 Mplayer 和 Qt 的音乐播放器

# 实现功能
- 音乐列表显隐
- 音乐播放和歌词定位显示
- 双击歌曲播放、播放和暂停、下一首、上一首
- 歌曲和音量进度条拖拽
- 音量图标一键静音和恢复
- 列表播放、随机播放、单曲循环以及模式组合
- ...
